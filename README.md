# Newsletter Agent
Agente automatizado que procesa newsletters de IA (AlphaSignal, TLDR),
genera resúmenes en español y crea contenido listo para publicar en redes sociales.

## ¿Qué hace?
- Lee correos de newsletters automáticamente
- Traduce y resume el contenido con IA
- Genera visualizaciones e infografías
- Publica o prepara posts para redes sociales

## Stack tecnológico
- Python 3.11+
- Gmail API
- Anthropic API (Claude)
- matplotlib / Pillow
